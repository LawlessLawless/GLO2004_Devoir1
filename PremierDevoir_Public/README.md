Les cliniques Santé et vie offrent 2 services : médecin et radiologie.

## DÉMARRAGE DE L'APPLICATION

### Building

```bash
mvn clean install
```

### Lancer l'application

```bash
mvn exec:java
```
Par défaut, le serveur roule à l'adresse suivante `http://localhost:9090`
